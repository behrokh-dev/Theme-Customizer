# 🎨 Theme Customizer

A responsive **Theme Customizer** built with **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**. This project demonstrates how to use **Local Storage** to save user preferences, allowing the selected theme settings to persist even after refreshing or reopening the browser.

## 💡 How It Works

1. Select your preferred theme settings.
2. The selected values are stored in the browser's Local Storage.
3. When you revisit or refresh the page, your saved preferences are automatically applied.
